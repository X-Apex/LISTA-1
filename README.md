# 🚀 Ejercicios de Programación de la lista 1

Este repositorio reúne una **lista de 25 ejercicios para principiantes**, pensada para quienes recién comienzan a programar.  
Cada ejercicio busca reforzar conceptos básicos y fomentar la práctica constante, lo que ayudará a mejorar la lógica y el dominio de distintos lenguajes.

El objetivo es mostrar cómo los mismos problemas pueden resolverse en **Java**, **Python** y **C++**, reforzando la capacidad de adaptación y permitiendo comparar la sintaxis y particularidades de cada lenguaje.  
De esta manera, podrás experimentar con ejemplos concretos y descubrir nuevas formas de aplicar la lógica de programación.

---

## 📘 Java
Lenguaje que destaca por su portabilidad y robustez, siendo ampliamente utilizado en entornos académicos y profesionales.  

---

## 🐍 Python
Un lenguaje versátil y fácil de leer, ideal para dar los primeros pasos en programación y también para proyectos más avanzados.  

---

## ⚙️ C++
Un lenguaje que sobresale por su alto rendimiento y control de memoria, perfecto para sistemas exigentes como videojuegos y software embebido.
